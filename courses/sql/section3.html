Introduction to SQL

Section 2: Filtering Data

Learn how to filter data using SQL queries.

Filtering Commands

Here are some SQL commands for filtering data:

-- Select records where the age is greater than 30
SELECT * FROM users WHERE age > 30;

-- Select records where the name starts with 'J'
SELECT * FROM users WHERE name LIKE 'J%';

-- Select records where the email contains 'example'
SELECT * FROM users WHERE email LIKE '%example%';
