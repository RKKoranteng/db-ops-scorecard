Introduction to SQL

Section 1: What is SQL?

SQL (Structured Query Language) is a standard language for managing and manipulating databases.

Basic SQL Commands

Here are some basic SQL commands you can run:

-- Select all records from a table
SELECT * FROM users;

-- Insert a new record into a table
INSERT INTO users (name, email) VALUES ('John Doe', 'john@example.com');

-- Update a record in a table
UPDATE users SET email = 'johndoe@example.com' WHERE id = 1;

-- Delete a record from a table
DELETE FROM users WHERE id = 1;
